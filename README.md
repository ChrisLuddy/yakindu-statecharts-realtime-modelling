# yakindu-statecharts-realtime-modelling
YAKINDU Statechart models of four real-time systems: an open choice controller, an intruder alarm with partial and full arming modes, a coffee machine dispenser handling Americano, Latte and Cappuccino, and an air fryer with meal-specific temperature and timing logic. Built using YAKINDU Statechart Tools with simulation and code generation.
